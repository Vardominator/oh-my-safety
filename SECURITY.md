# Security Policy

To report a vulnerability in oh-my-safety, please use GitHub's private
vulnerability reporting (**Security → Report a vulnerability**) or contact the
maintainer privately — **do not** open a public issue for security matters.

Full details, scope, and the security properties we commit to (no network calls
from security checks, no telemetry, least privilege) are in
[docs/security-policy.md](docs/security-policy.md).
